What do the 9 Folders Contain?

 

Compatible Files

This Folder contains the files that were tested and can be encrypted and compressed.  Most of the other files

not present here  also will work.

 

DeC

This folder contains the Visual C++ Files and workspace and CPP files that give the coding for the main

decrypter algorithm. It also contains the folder called Decrypter which is Dll version of the decrypter

algorithm .

 

EnC

This folder contains the Visual C++ Files and workspace and CPP files that give the coding for the main

Encrypter algorithm. It also contains the folder called Encrypter which is Dll version of the Encrypter

algorithm .

 

DS

Here we have the coding for the Stack data structure used in the encrypter and decrypter algorithm to reverse

sets of bytes. Remeber the Stack has been implemented using the SmartNode class which is also present in the same

directory.  Read "How this software was developed." section in the index.Contains all VC++ files. Can be opened

 easily with any system with VC++.

 

Rand

This is the coding for the random number generator used to generate random numbers for encryption and decryption

 algorithms. Contains all VC++ files. Can be opened easily with any system with VC++.

 

Readme

This folder contains the HTML help and associated files which help in using the software. And it helps develop similar

 softwares and help in understanding this one. Contains all image files and linking information and induvidual pages like this

 one.

 

VB1

Front end of the sofware. Whatever the user sees in windows is all in VB. Makes heavy use of Windows API.

Contains all VB support files. Can be opened easily with any system with VB. Contains all form controls and can be easily

 opened from VB for editing, upgrading, customising and developing the software.

 

DecCopy

This folder contains the copy of the decryption software seperately so as to enable the full version of the software may be able

 to inmplement the option "Place a copy of the decryption software along with the output files." This helps in distribution and

in providing maximum comfort to both the encryptor person and the target person who has to decrypt.

 

Directory

Coding and Library files for various inbuilt functions of the full version of the software is present here. This provides Fault

 Tolerance with codings for DATChecker, Datkey, KeyExt, Direc.

DATChecker    -    Cheks whether the extension of a string is .dat or not.

Datkey              -    Converts a string with extension .dat  into .key.

KeyExt              -    Inserts a string with "<foldername>/" into a target string with .dat extension. 

Direc                  -    Gives the directory of a string containing full address with extension of a file by deleting from last untill a

                                slash is encountered.

 

    Most of the codings are self explanatory and can be easily understood. So Comments are not given.
