MINI PROJECTS

 

1. GAP2P – A peer to peer optimizing solution.                                                       Team Size: 2

 

            Peer to Peer software designed to facilitate Network Administrators optimized to work on LAN, WAN & WLANs. The GA stands for “Give Away” which is the principle that each peer performs. All peers “seed” their part of the whole resource – functionality comparable to BitTorrent.

2. Online Encryption Software “CCura+”                                                                  Team Size: 2

     

            The project aims at developing a closed source encryption algorithm, over standard DES or 3DES encryption to satisfy special needs of total security assurance. It involves an Object-Stack whose size is determined by a pseudo-random number generator. Two such Stacks will work in tandem to re-order encrypted information, thus making it very, very difficult to break.

3. An FPS 3-d game using C# 2.0 and Release Aug 2006 - Managed DirectX

 

            The game is driven by the Idle Event – which is raised continuously and its handler function. Other user-caused events produce changes in variables, which are interpreted and realized by the idle event handler. It has 7 textures, 1 sound, and reads the map of the 3D world from a textfile.

 

4. TecMaze: A maze that tests your technical knowledgege                                    

                           

            This VB.NET standalone app has a map designer, which generates bitmap images, and map-metadata. Can also generate meta-data for existing images. “Meta-data” refers to a grid of accessible and inaccessible locations, start/stop positions, etc. These maps can later be played, with obstacles asking questions. Originally a .Net 3.5 App, but downgraded to 2.0, for size & install-time reduction.

5. DIC-Complete: A WPF standalone VB.NET Interest Calculator                                                                  Team Size: 2

            This is a very rich UI, simple Daily Interest Calculating application, for small businesses involving transactions. Split button, Glow button, smooth gradient, smooth animation are implemented
